# Compatibility documentation pointer

The visible product name is now **newsScrapper**. This filename remains only so
existing bookmarks and deployment notes do not break.

Use [`NEWSSCRAPPER_WINDOWS_GUIDE.md`](NEWSSCRAPPER_WINDOWS_GUIDE.md) for the
current Windows setup, certificate handling, model installation, architecture,
source preflight, troubleshooting, and operational validation instructions.

Internal identifiers such as the `signalroom` Python package, `/api/signalroom`
bridge, `SIGNALROOM_*` environment variables, browser-storage keys, and
serialized state remain intentionally compatible.
